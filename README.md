# Dashboard ERP Mustika

Selamat datang di repositori Mustika ERP System. 
Aplikasi ini adalah Sistem Perencanaan Sumber Daya Perusahaan (ERP) internal yang dibangun khusus untuk CV. Mustika (Mustika Advertising) guna mendigitalisasi alur kerja operasional perusahaan.

Sistem ini dirancang untuk mengelola seluruh siklus workflow mulai dari penerimaan SPS, Survey, Desain, Quotation, Produksi, hingga Pemasangan, dilengkapi dengan manajemen master harga (Pricelist) dan Hak Akses Pengguna (Role-Based Access Control).

Repositori Utama: https://github.com/Danzzpo/Dasboard_ERP_Mustika

---

## Teknologi yang Digunakan (Tech Stack)

Proyek ini dibangun menggunakan arsitektur modern dengan spesifikasi berikut:
* Backend: Laravel 12
* Frontend: Vue.js 3 (SFC & Composition API)
* Styling: Tailwind CSS
* Starter Kit: Laravel Breeze (Inertia.js)
* Database: MySQL

---

## Persyaratan Sistem (System Requirements)

Sebelum menjalankan proyek ini, pastikan komputer atau laptop Anda sudah terinstal perangkat lunak dengan versi minimal sebagai berikut:
* PHP: Minimal versi 8.2
* Node.js: Minimal versi 20.x
* NPM: Tersedia otomatis saat menginstal Node.js
* Composer: Versi terbaru (versi 2.x)
* Database: MySQL (Bisa menggunakan XAMPP)
* Tools Database: Direkomendasikan menggunakan aplikasi DBeaver untuk manajemen database.
* Git: Versi terbaru
